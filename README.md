# riot-api-lol-tracker

How to use it :

1. Go to https://developer.riotgames.com/ 

2. Create a new account or login with your riot account

3. Then generate a new api Development API Key and copy it.

4. Create an .env file and create the API_KEY variable and paste the api key

Example:
```

API_KEY=APIkeyEXAMPLE123

```
